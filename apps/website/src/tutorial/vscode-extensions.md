## VS Code 插件推荐清单

为了提高 Python、前端（Vue/React）、FastAPI 以及 `monorepo` 项目的开发效率，我们整理了一套 **VS Code** 插件组合，覆盖了以下几个方面：

## 主题与界面

- [Atom One Dark Theme](https://marketplace.visualstudio.com/items?itemName=akamud.vscode-theme-onedark) — One Dark 主题
- [vscode-icons-mac](https://marketplace.visualstudio.com/items?itemName=wayou.vscode-icons-mac) — Mac 风格文件图标
- [Rainbow Brackets](https://marketplace.visualstudio.com/items?itemName=2gua.rainbow-brackets) — 彩虹括号，高亮显示不同嵌套层级

## 代码运行与调试

- [Code Runner](https://marketplace.visualstudio.com/items?itemName=formulahendry.code-runner) — 快速运行和调试代码
- [Python Debugger](https://marketplace.visualstudio.com/items?itemName=ms-python.debugpy) — Python 调试工具
- [WSL](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-wsl) — 在 WSL 环境下打开、编辑和调试代码

## Python 开发

- [Python](https://marketplace.visualstudio.com/items?itemName=ms-python.python) — Python 官方核心插件
- [Pylance](https://marketplace.visualstudio.com/items?itemName=ms-python.vscode-pylance) — Python 智能语言服务器
- [Mypy Type Checker](https://marketplace.visualstudio.com/items?itemName=ms-python.mypy-type-checker) — Python 静态类型检查工具
- [Python Environments](https://marketplace.visualstudio.com/items?itemName=ms-python.vscode-python-envs) — 管理 Python 虚拟环境
- [Python Test Explorer for Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=LittleFoxTeam.vscode-python-test-adapter) — 测试集成插件
- [python-snippets](https://marketplace.visualstudio.com/items?itemName=cstrap.python-snippets) — Python 常用代码片段
- [Python Type Hint](https://marketplace.visualstudio.com/items?itemName=njqdev.vscode-python-typehint) — Python 类型提示自动补全
- [Ruff](https://marketplace.visualstudio.com/items?itemName=charliermarsh.ruff) — Python linter / 格式检查工具
- [autoDocstring - Python Docstring Generator](https://marketplace.visualstudio.com/items?itemName=njpwerner.autodocstring) — 自动生成 Python docstring 注释

## 前端开发与 Vue

- [Vue (Official)](https://marketplace.visualstudio.com/items?itemName=Vue.volar) — 官方 Vue 插件
- [Vue VSCode Snippets](https://marketplace.visualstudio.com/items?itemName=sdras.vue-vscode-snippets) — Vue 代码片段
- [UnoCSS](https://marketplace.visualstudio.com/items?itemName=antfu.unocss) — 原子 CSS 工具插件
- [Naive UI IntelliSense](https://marketplace.visualstudio.com/items?itemName=tu6ge.naive-ui-intelligence) — Naive UI 自动提示
- [CSS Variable Autocomplete](https://marketplace.visualstudio.com/items?itemName=vunguyentuan.vscode-css-variables) — CSS 全局变量自动补全
- [Stylelint](https://marketplace.visualstudio.com/items?itemName=stylelint.vscode-stylelint) — CSS/SCSS/Less 风格和语法检查

## 代码规范与格式化

- [Prettier - Code formatter](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) — 代码格式化
- [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) — JavaScript / TypeScript 代码检查
- [Document This](https://marketplace.visualstudio.com/items?itemName=oouo-diogo-perdigao.docthis) — JS/TS 自动注释
- [Code Spell Checker](https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker) — 拼写检查
- [TODO Highlight](https://marketplace.visualstudio.com/items?itemName=wayou.vscode-todo-highlight) — 高亮显示 TODO/FIXME/NOTE

## 远程开发与 Git

- [Remote Repositories](https://marketplace.visualstudio.com/items?itemName=ms-vscode.remote-repositories) — 直接在 VS Code 中访问远程仓库
- [Git Graph](https://marketplace.visualstudio.com/items?itemName=mhutchie.git-graph) — Git 可视化管理
- [GitHub Repositories](https://marketplace.visualstudio.com/items?itemName=GitHub.remotehub) — 在 VS Code 直接操作 GitHub 仓库
- [Dev Containers](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers) — 容器化开发环境
- [Container Tools](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-containers) — Docker / Podman 容器管理
- [Docker DX](https://marketplace.visualstudio.com/items?itemName=docker.docker) — Docker 插件

## 文档与可视化

- [Markdown Preview Enhanced](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced) — Markdown 实时预览
- [i18n Ally](https://marketplace.visualstudio.com/items?itemName=Lokalise.i18n-ally) — 国际化翻译支持
- [Icônes](https://marketplace.visualstudio.com/items?itemName=afzalsayed96.icones) — 查找可用图标
- [Iconify IntelliSense](https://marketplace.visualstudio.com/items?itemName=antfu.iconify) — 编辑器中显示 Icon
- [javascript console utils](https://marketplace.visualstudio.com/items?itemName=whtouche.vscode-js-console-utils) — JS console.log 清理工具
