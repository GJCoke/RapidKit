# @rapidkit/frontend

## 1.3.0

### Minor Changes

- ### Feature
  - Renamed the `web` application to `frontend`
  - Integrated `openapi-typescript` and added a corresponding Vite plugin to automatically regenerate types on file
    changes
  - Added `ApiRequest` and `ApiResponse` type helpers to simplify usage of the schemas generated by openapi-typescript
  - Refactored the `Service` type definitions out of `App` for a cleaner structure and better reusability
